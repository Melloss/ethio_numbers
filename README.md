## Getting started
you can add the package using the following command

```
flutter pub add ethio_numbers
```
or

```
dart pub add ethio_numbers
```


## Usage

you can convert using `.toAmharicText()` extension.


```dart
print(7023.toAmharicText()); //output: ሰባት ሺ ሀያ ሶስት
```
